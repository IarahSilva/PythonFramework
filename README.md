## 📖 Sobre o Curso

Este curso tem como objetivo ensinar os fundamentos da linguagem Python e o desenvolvimento de aplicações web utilizando o framework Django.

---

## 🎯 Objetivos

- Aprender Python do básico ao avançado.
- Desenvolver lógica de programação.
- Criar aplicações web com Django.
- Trabalhar com banco de dados.
- Desenvolver projetos completos.

---

## 🛠️ Tecnologias

- Python 3.x
- Django
- SQLite
- HTML5
- CSS3
- Bootstrap
- Git e GitHub

---

## 📚 Conteúdo Programático

### Módulo 1 - Python
- Variáveis
- Operadores
- Estruturas de decisão
- Estruturas de repetição
- Funções
- Listas, Tuplas e Dicionários

### Módulo 2 - POO
- Classes
- Objetos
- Herança
- Polimorfismo

### Módulo 3 - Django
- Models
- Views
- Templates
- URLs
- Formulários

### Módulo 4 - Banco de Dados
- SQLite
- Django ORM
- CRUD

### Módulo 5 - Projeto Final
- Desenvolvimento de um sistema completo
